# ABDOULAYE STORE 
Site e-commerce frontend développé en HTML, CSS et JavaScript.

## Fonctionnalités
- Ajout de produits au panier
- Panier interactif 
- Suppression de produits
- Gestion des quantités
- Persistance des données avec LocalStorage
- Calcul automatique du total

## Structure du projet
aasets/images => on y trouve les images du projets; 
css/ => le code css;  
js/ => le code js;  
index.html : la page principale du projet
